<inner> boxIntersect(aX, aY, aW, aH, bX, bY, bW, bH)

Generic intersect check with square bounding box.
Assumes x: 0, y: 0 is the top left corner
for both objects and the screen

Parameters:
Name Type Description
aX float

The x position of the first object.
aY float

The y position of the first object.
aW float

The width of the first object.
aH float

The height of the first object.
bX float

The x position of the second object.
bY float

The y position of the second object.
bW float

The width of the second object.
bH float

The height of the second object.
Source:
  • physics.js, line 12
Returns:

True if there is an intersection.

Type
bool
Example
var rect1 = {
  x: -2,
  y: 1,
  width: 5,
  height: 5
};
var rect2 = {
  x: 2,
  y: 4,
  width: 10,
  height: 18
};
// returns true
DL_.boxIntersect(rect1.x, rect1.y, rect1.width, rect1.height, rect2.x, rect2.y, rect2.width, rect2.height);
