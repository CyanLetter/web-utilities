(static) boxIntersect(aX, aY, aW, aH, bX, bY, bW, bH) → {bool}

Generic intersect check with square bounding box. Assumes x: 0, y: 0 is the top left corner for both objects and the screen
Parameters:
Name Type Description
aX float The x position of the first object.
aY float The y position of the first object.
aW float The width of the first object.
aH float The height of the first object.
bX float The x position of the second object.
bY float The y position of the second object.
bW float The width of the second object.
bH float The height of the second object.
